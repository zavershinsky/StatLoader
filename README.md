# StatLoader - приложение для автоматизированной обработки статистической информации, получаемой из БД, и отправки результата по email.
# Создана для автоматизации рутины по сбору статистики и отправки Key Account Manager.
Приложение может работать как под управлением Windows, так и под управлением Linux. Протестирована работа приложения с Python 3.9.2 64-bit.

Инсталляция под Windows:
1) Установить pip_requirements.txt
2) Создать каталог C:\PyServices\[AppName], где AppName - название приложения
3) В файле win_service.py указать название приложения из п.2 в качестве значения переменной _svc_name_
4) Перенести файл настроек по умолчанию InvoiceStatistics.yml в каталог, созданный в п.2, и переименовать по названию приложения
5) Запустить под администратором: python.exe win_service install
6) В результате получим службу Windows, которая управляется через services.msc

Под Linux инсталляции не требуется, запуск производится через run.sh, остановка - shutdown.sh
